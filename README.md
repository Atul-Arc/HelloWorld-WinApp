# HelloWorldWinApp

A simple Windows Forms desktop application built using **C# and .NET Framework 4.8** that demonstrates core Windows Forms common controls and event handlers.

## Features

### Greeting Generator
- **Text Input**: Enter a name in the text box to personalize your greeting
- **Dynamic Greeting**: Displays "Hello, [Name]" with customizable punctuation
- **Default Fallback**: Shows "Hello, World" if no name is provided
- **Exclamation Toggle**: Optional checkbox to add excitement ("!!!" or ".")

### Interactive Controls
- Text input field (`txtName`) for user name entry
- Greeting label (`lblGreeting`) for dynamic output display
- Exclamation checkbox (`chkExclamation`) to customize punctuation
- Description label (`lblDescription`) with application details
- GitHub link button to visit the developer's profile

## Getting Started

1. Clone the repository
2. Open `HelloWorldWinApp.sln` in Visual Studio
3. Build the solution
4. Run the application
5. Enter a name and click the button to generate a greeting

## Event Handlers

- `button1_Click`: Processes user input and generates the greeting message
- `linkLabel1_LinkClicked`: Opens the developer's GitHub profile
- `Form1_Load`: Initializes the description text with application details
